Become an Architect.

Mithl OS is built by people like you. Whether you write low-level C code, design icons, or just love testing new software, there is a place for you here.

View Open Issues

You can do these or Simple or Further or What you know to do or Beyond these:

Kernel Hackers

Help us implement the VFS (Virtual File System) and improve our Paging algorithms in C & Assembly.

Rustaceans

We are porting critical drivers to Rust for memory safety. Help us rewrite the PS/2 and ACPI drivers.

UI Designers

Mithl OS needs a unique identity. Help design icons, wallpapers, and window themes.

App Ecosystem

We need an App Store! Help us build the package manager and create native apps (Calculator, Media Player, Games) for the OS.

How to Contribute

  • 1. Fork the repository on GitHub.
  • 2. Clone it to your local machine.
  • 3. Create a Branch (e.g., `git checkout -b feature/new-driver`).
  • 4. Commit your changes and push.
  • 5. Open a Pull Request!
